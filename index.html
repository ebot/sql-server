sql-server by ebot

Fast and easy Microsoft Sql Server querying.

This library makes it easy to query SQL Server from a ruby script: 

 1 require 'rubygems'
 2 require 'SqlServer'
 3 
 4 db = SqlServer.new( { :host => '10.2.1.135', 
 5                       :user_name => 'people_test', 
 6                       :password => 'password', 
 7                       :database => 'people' } )
 8 
 9 qry = 'SELECT first, middle, last, suffix from names'
10 
11 names = db.query('SELECT first, last, middle FROM names')
12 names.each { |name| puts "#{name['last']}, #{name['first']} #{name['middle']}" }
